# 实用工具集（原 windows-utils 合并）

本目录包含从 `windows-utils` 仓库合并的工具，作为 win-optimizer 的补充功能模块。

## 工具列表

| 工具 | 语言 | 说明 |
|------|------|------|
| C盘清理工具 | Python + Tkinter GUI | 扫描并清理系统临时文件、缓存、回收站等 |
| 剪贴板修复工具 | Python + BAT | 重启剪贴板服务，修复复制粘贴失效问题 |
| 网络重置工具 | Python + BAT | 一键重置网络设置，解决网络连接问题 |
| 移除内部标记 | HTML | 浏览器工具，移除内部标记/水印 |

## 快速启动

从主菜单 `StartAll.bat` 选择对应工具，或直接运行各工具目录下的 BAT 文件。

## 与优化脚本的关系

- `scripts/` 目录：PowerShell 系统优化脚本（服务、启动项、电源计划等）
- `tools/` 目录：Python 实用工具（清理、修复、重置）

两者互补：优化脚本提升系统性能，实用工具解决具体问题。
